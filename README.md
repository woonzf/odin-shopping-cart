# Mizu Store

### [Demo](https://woonzf.github.io/odin-shopping-cart/)

### About:

A mock shopping cart application consists of store page, product page and cart page using [Fake Store API](https://fakestoreapi.com/).

A project for The Odin Project: React Course's [Project: Shopping Cart](https://www.theodinproject.com/lessons/node-path-react-new-shopping-cart).

### Tech stack used:

TypeScript, React, Tailwind, Lucide React, Prettier, Vitest
